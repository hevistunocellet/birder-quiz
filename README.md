# Birding Quiz

Un test interactiu per descobrir quin tipus d'ornitòleg ets.  
Respon les 20 preguntes amb una escala de l'1 al 5 i al final obtindràs el teu perfil amb descripció i imatge.

## Perfils
- Lister
- Km.0
- Fotògraf
- Caçador de Pokémons
- Etòleg
- Viatger
- Natura
- Casual

## Com funciona
1. Obre la pàgina a GitHub Pages.
2. Respon les preguntes.
3. Clica "Finalitzar quiz".
4. Llegeix el teu perfil i mira la imatge.
